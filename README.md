## 📌 Module Overview

### `main.py`
Acta as the main entry point of the application.

### `dashboard.py`
Handles dashboard rendering, charts, and visualization components.

### `analytics.py`
Performs financial calculations, analysis, and data processing.

### `database.py`
Responsible for database connection, queries, and secure data storage.

### `validation.py`
Contains input validation and data verification functions.

### `utils.py`
Reusable Helper functions used across the project.

### `reports.py`
Generates financial summaries, reports, and analytics.

Clone the repository:

```bash

```

Move into the project directory:

```bash
cd finance-dashboard
```

Run the prjoect:

```bash
python main.py
```
